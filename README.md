# PIB-Configuration-Tool
A Configuration Tool for Pigs in a Blanket

#### Note
The custom resolution mode configuration works as so:

|Mode|Forced Resolution Mode|Sharpscale Required|
|:-:|:-:|:-:|
|0|OFF|Up to Developer|
|1|960X544|no|
|2|720X408|no|
|3|640X368|no|
|4|480X272|no|
|5|1280X720|Vita|
|6|1920X1080|Vita|
